# 面试录 - iOS 求职记录管理 App

一款简洁、实用的 iOS 求职记录管理应用，帮助求职者追踪和管理面试进度。

## ✨ 功能特性

### 📋 求职记录管理
- 添加、编辑、删除求职记录
- 记录公司名称、岗位、投递日期、工作地点、薪资范围
- 支持备注信息

### 🎯 面试轮次追踪
- 添加多轮面试记录（一面、二面、三面、交叉面、HR面等）
- 支持自定义轮次名称
- 记录面试官、面试结果、面试反馈
- 支持 Offer 轮次

### 📊 流程进度可视化
- 动态流程进度条，直观显示求职进度
- 根据面试轮次自动更新进度
- 颜色标识：进行中（橙色）、已通过（绿色）、已拒绝（红色）

### 🏷️ 智能状态标签
- 右上角状态标签：已通过 / 流程中 / 已拒绝
- 根据面试结果自动判断状态
- Offer 显示 ✉️ 信封图标
- 面试失败显示 ❌ 红色叉叉

### 📱 快捷操作
- 列表页左滑显示"未通过"和"删除"选项
- 投递阶段被挂可直接标记为"未通过"
- 无需先添加面试轮次

## 🛠️ 技术栈

- **UI 框架**：SwiftUI
- **数据持久化**：SwiftData (iOS 17+)
- **最低支持版本**：iOS 17.0
- **开发语言**：Swift 5.9+

## 🚀 快速开始

### 环境要求
- Xcode 15.0+
- iOS 17.0+
- macOS 14.0+

### 安装步骤
1. 克隆仓库
   ```bash
   git clone https://github.com/ori-yin/interview-tracker.git
   ```

2. 打开项目
   ```bash
   cd interview-tracker
   open 面试录.xcodeproj
   ```

3. 选择模拟器或真机，点击运行

## 📖 使用说明

### 添加求职记录
1. 点击右上角 "+" 按钮
2. 填写公司名称、岗位名称（必填）
3. 选择投递日期
4. 填写工作地点、薪资范围（可选）
5. 点击"保存"

### 添加面试轮次
1. 进入求职记录详情页
2. 点击"添加面试记录"
3. 选择轮次名称（一面、二面、HR面等）
4. 选择面试日期
5. 选择本轮结果（待定、通过、未通过、已取消）
6. 填写面试反馈（可选）
7. 点击"保存"

### 标记未通过
- **方式一**：在列表页左滑，点击"未通过"
- **方式二**：在详情页编辑，修改状态为"未通过"
- **方式三**：在面试轮次中，选择结果为"未通过"

## 🔧 核心红线

本项目严格遵守以下原则：

1. **纯本地运行**：所有数据存储在用户手机本地
2. **禁止云端数据库**：不使用 Firebase、CloudKit 等
3. **禁止用户系统**：不添加登录/注册功能
4. **禁止网络请求**：不使用 Alamofire 等网络库
5. **原生 UI**：仅使用 SwiftUI，不使用 UIKit
6. **无第三方库**：不使用第三方 UI 库
